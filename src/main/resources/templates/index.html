Bem-vindo ao User Service

Esta é uma aplicação desenvolvida em Spring Boot que oferece um sistema completo de gerenciamento de usuários com arquitetura multi-tenant, permitindo o isolamento de dados entre diferentes organizações ou clientes.

Funcionalidades disponíveis:
  • CRUD completo de usuários com validações customizadas
  • Isolamento total de dados entre diferentes tenants
  • Interface web responsiva com gerenciamento de tenants
  • Segurança com hash BCrypt para senhas
  • API REST completa para integrações
  • Sessão por tenant com fallback automático
